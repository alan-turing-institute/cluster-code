# Run tests

## Using `fab` (local only)

To run unit tests using `fab`, run:

```bash
fab standalone.setup standalone.pytest
```

You should see:

```bash
============================= test session starts ==============================

```

## Using `pytest`

To run unit tests using `pytest`, run:

```bash
cd standalone
pytest
```
